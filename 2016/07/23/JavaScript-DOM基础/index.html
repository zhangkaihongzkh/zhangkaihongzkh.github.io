JavaScript-DOM基础

发表于 2016-07-23   |   分类于 JavaScript基础篇   |   | 阅读量 次

摘要:

DOM介绍

DOM中的三个字母
D（文档）可以理解为整个web加载的网页文档 O（对象）可以理解为类似window对象之类的东西，这里我们指的是document对象 M（模型）可以理解为网页文档的模型结构

节点

在加载HTML页面时，Web浏览器会生成一个树形结构，用来表示页面的内部结构
而DOM将这种树形结构理解为由节点组成

从上图我们知道html没有父标签和兄弟标签，所以html标签为根标签
而head标签是html标签的子标签 meta和title是兄弟标签。
如果把每个标签都当作一个节点的话，这样的结构就形成了一棵节点树
PS：我们将标签也称为元素

节点种类

节点种类：元素节点，文本节点，属性节点

查找元素

W3C给我们提供了定位节点的方法和属性

查找元素的方法

方法说明
getElementById(‘#id’)获取特定ID元素的节点
getElementsByTagName(‘tagName’)获取特定相同元素的节点列表
getElementsByName(‘name’)获取相同名称的节点列表
getElementsByClassName(‘.class’)获取相同类名的节点列表
getAttribute(‘attr’)获取特定元素节点属性的值
setAttribute(‘attr’,’value’)设置特定元素节点属性的值
removeAttribute(‘attr’)移除特定元素节点属性

元素节点的属性

元素节点的属性

属性说明
tagName元素节点的标签名
innerHTML元素节点里的内容，非W3C标准

DOM节点

node节点属性

  • nodeName
  • nodeType
  • nodeValue

    node节点属性

    节点类型nodeNamenodeTypenodeValue
    元素元素名称1null
    属性属性名称2属性值
    文本#text3文本值(不包括html)

层次节点属性

层次节点属性，我们主要分为 父与子节点 和 兄弟节点 两种

层次节点属性

属性说明
childNodes获取当前元素节点所有子节点
firstChild获取当前元素节点第一个子节点
lastChild获取当前元素节点最后一个子节点
ownerDocument获取该节点的文档根节点–相当于document
parentNode获取当前节点的父节点
previousSibling获取当前节点的前一个兄弟节点
nextSibling获取当前节点的后一个兄弟节点
attributes获取当前节点所有属性节点的集合

innerHTML与nodeValue的区别

  • 在获取文本节点的时候，无法用innerHTML这个属性输出文本内容的。innerHTML必须获取到元素节点 才能输出包含的文本

  • 在赋值的时候nodeValue会把包含在文本的HTML转义成特殊字符，从而达到 纯文本 的效果

节点操作

DOM不仅仅可以查找结点 还可以创建节点，复制节点，插入节点，删除节点，替换节点

节点的操作说明

方法说明
write(st)把任意字符串插入到文档中
createElement(element)创建一个元素节点
appendChild(child)将新节点添加到子节点列表的末尾
createTextNode(textNode)创建一个文本节点
insertBefore(p,box)将新节点插入在前面
replaceChild(p,box)将新节点替换旧节点
cloneNode(node)复制节点
removeChild(node)移除节点
