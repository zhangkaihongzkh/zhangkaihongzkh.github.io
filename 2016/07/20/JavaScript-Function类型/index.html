函数的内部属性

在函数内部，有两个特殊的对象：arguments 和 this。

  • arguments 是一个类数组对象，包含传入函数中的所有参数，主要用于保存函数的参数。这个对象还有一个 callee 的属性，该属性是一个指针，指向拥有这个arguments对象的函数

对于阶乘函数一般要用到递归，如果函数名更改，函数名内部也要更改，这时候 callee 属性就发挥作用了

  • 对于函数内部另一个对象特殊对象 this
    this引用的是函数具体执行操作的对象，即函数调用语句具体的作用域
    当在全局作用域时，this对象引用就是window

函数属性和方法

ECMAScript中函数是对象，因此函数也有属性和方法。其中属性有两个，分别为 length 和 prototype

length 表示接受函数希望接收命名参数的个数

prototype 是用于所有实例方法真正的所在，（将在面向对象一节终点阐述） 而其下有两个方法：apply() 和 call()

每个函数都包含这两个非继承而来的方法。
用途是在特定的作用域中调用函数，即在函数体内设置this对象的值

  • apply

而apply与call的不同在于apply接收可以是arguments和参数数组 call传递的参数只能逐个列举出来

  • call

事实上call和apply真正强大的地方在于其能扩充函数赖以运行的作用域

使用call()或者apply()来扩充作用域最大的好处就是不需要与方法发生任何 耦合关系 (就是互相关联的意思，扩展和维护会发生连锁反应)。也就是说o对象和sayColor方法不会有多余的关联操作
eg.o.sayColor = sayColor;
