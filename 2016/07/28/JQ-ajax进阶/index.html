JQ-ajax进阶

发表于 2016-07-28   |   分类于 JQ进阶篇   |   | 阅读量 次

摘要：1.加载请求 2.错误处理 3.请求全局事件 4.JSON和JSONP

加载请求

JQ提供了两个全局加载请求函数 $(document).ajaxStart 和 $(document).ajaxStop()
其中$下的选择器必须是document

如果想设置ajax请求时间 可以在$.ajax中增加 timeout:500
如果想取消全局 可以在$.ajax中增加 global:false

错误处理

在ajax()参数中增加error字段

并不一定每次请求都是成功的，这时候我们需要给用户提供错误信息指导

在ajax()函数后增加.error

同时也可以在ajax函数之后添加.error()

在ajax函数后增加.fail

全局请求事件

jq对ajax提供了许多全局方法，就像上面的.ajaxStart() .ajaxStop() .ajaxError()他们都属于全局请求时触发
除了这些，jq与js类似，也有在出发前后的全局事件

  • .ajaxSuccess(), 对应有一个局部方法.success()，请求成功时执行
  • .ajaxComplete(), 对应有一个局部方法.complete()，请求完成后注册一个回调函数
  • .ajaxSend(),没有局部方法，只有属性beforeSend，请求发送前要绑定的函数

$.post使用局部方法.success

$.post使用ajaxSuccess(event,xhr,settings)全局方法

全局事件的方法是所有ajax请求都会触发，一般要 绑定$(document)事件。而局部方法 是针对某个ajax

for in得到全局事件的参数中的属性

对于全局事件方法的参数，大部分为对象。对于这些对象有哪些属性方法能调用，可以通过遍历方法得到

$.post使用局部方法.complete

$.post使用ajaxComplete(event,xhr,settings)全局方法

JSON和JSONP

如果想跨域操作文件的话，我们就必须使用 JSONP。JSONP(JSON with Padding)是一个非官方的协议，它允许在服务器端集成 Script tags 返回至客户端，通过javascript callback 的形式实现跨域访问（这仅仅是 JSONP 简单的实现形式）
