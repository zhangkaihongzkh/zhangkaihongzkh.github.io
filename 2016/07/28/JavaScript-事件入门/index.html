JavaScript-事件入门

发表于 2016-07-28   |   分类于 JavaScript基础篇   |   | 阅读量 次

摘要：1.事件介绍 2.内联模型 3.脚本模型 4.事件处理函数

JavaScript事件是由访问 Web页面的用户引起的一系列操作， 例如： 用户点击。 当用户执行某些操作的时候，再去执行一系列代码。

事件介绍

JavaScript有三种事件模型：内联模型 、 脚本模型和 DOM2模型

内联模型

这种模型是最传统接单的一种处理事件的方法。在内联模型中， 事件处理函数是 HTML标签的一个属性， 用于处理指定事件。 虽然内联在早期使用较多， 但它是和 HTML混写的，并没有与 HTML分离。

PS：函数不得放到 window.onload里面，这样就看不见了。

脚本模型

由于内联模型违反了 HTML与 JavaScript代码层次分离的原则。为了解决这个问题，我们可以在 JavaScript中处理事件。这种处理方式就是脚本模型。

PS：通过匿名函数，可以直接触发对应的代码。也可以通过指定的函数名赋值的方式值

事件处理函数

JavaScript可以处理的事件类型为：鼠标事件、键盘事件、 HTML事件。

JavaScript事件处理函数及其使用列表








onabort
图像
当图像加载被中断时


onblur
窗口、框架、所有表单对象
当焦点从对象上移开时


onchange
输入框，选择框和文本区域
当改变一个元素的值且失去焦


onclick
链接、按钮、表单对象、图像映射区域
当用户点击时


ondblclick
链接、按钮、表单对象
当用户双击对象时


ondragdrop
窗口
当用户将一个对象拖放到浏览器窗口时


onError
脚本
当脚本发生语法错误时


onfocus
窗口、框架、所有表单对象
当单击鼠标或者将鼠标移动聚焦到窗口或框架时


onkeydown
文档、图像、链接、表单
当按键被按下时


onkeypress
文档、图像、链接、表单
当按键被按下然后松开时


onkeyup
文档、图像、链接、表单
当按键松开时


onload
主题、框架集、图集
文档或图像加载后


onunload
主题、框架集
文档或框架卸载后


onmouseout
对象
当鼠标移出目标时


onmouseover
对象
当鼠标移到目标时


onmousemove
对象
当鼠标移动时


onmove
窗口
当浏览器窗口移动时


onresize
窗口
当浏览器窗口变化时


onreset
表单复位按钮
点击表单的reset按钮


onselect
表单元素
当选择一个表单对象时


onsubmit
表单
当发送表格到服务器时

事件处理函数 影响的元素 何时发生

PS： 所有的事件处理函数都会都有两个部分组成， on+ 事件名称， 例如click事件的事件处理函数就是： onclick。 在这里， 我们主要谈论脚本模型的方式来构建事件， 违反分离原则的内联模式，我们忽略掉。

对于每一个事件， 它都有自己的触发范围和方式， 如果超出了触发范围和方式， 事件处理将失效。
