添加categories、tags等页面

我们在github page上不难发现 自己博客的categories还有tags页面是空的,打开为404.这时我们需要新建这两个页面

  • 添加categories页面
    在你的blog文件夹下git bash

在source文件夹下就会有categories文件夹，进入并编辑index.md,增加type: “categories”字段.然后进入主题的_config.yml中，menu下的#categories注释去掉
这时categories就可以打开了

  • 同理,about tags页面也是一样

1.$ hexo new page “对应的名字”
2.进入source\对应文件夹\index.md 增加对应字段
3.在配置文件下的 _config.yml中menu 将对应页面#去掉

自此，你的blog页面也就基本完成啦
