云主机搭载LAMP部署PHP项目

发表于 2018-02-28   |   分类于 躺过的坑   |   | 阅读量 次

摘要:
腾讯云主机搭载LAMP环境并部署PHP项目

最近尝试将自己项目部署上腾讯云服务器，现在记录部署流程以及犯过的错误。

犯过的错误

对ssh认证理解不够透彻

现在网上流行的基本都是ssh认证，包括工具winSCP、putty等工具。现在简述SSH认证过程。
1.第一次登录服务器时，将服务器的公钥传递过来，本地计算机进行存储，下次传递数据时将使用这个公钥对数据进行加密。
2.服务器接收到利用自己公钥加密的数据后，使用自己的私钥才能解开

数据库连接设置

由于是MySQL数据库配置的默认网关是127.0.0.1(本机地址)，而且MySQL默认是关闭远程访问权限的。当我将项目部署到服务器上时，连接的是公网地址，所以无法连接到数据库。这时候需要
1.修改mysql数据库的user表，将host字段的值改为%，这样在任何客户端机器上能以root用户登录到mysql服务器
2.修改mysql配置文件vi /etc/mysql/mysql.conf.d/mysqld.cnf 将bind-address = 127.0.0.1注销​.
这样就是实现远程登录mysql，同时公网也能连接上数据了
