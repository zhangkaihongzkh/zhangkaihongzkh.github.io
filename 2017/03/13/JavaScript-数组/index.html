数组的定义

数组是按次序排列的一组值,每个值的位置都有编号，从0开始，整个数组用方括号表示

数组的本质

本质上，数组属于一种特殊的对象 typeof 运算符返回的类型是 object

typeof 可以确定数组就是数组的类型就是对象

数组的特殊性在于，他的键名是先找次序排列的一组整数 （0,1,2……）

Object.keys方法会返回所有的键名。所以数组的键名就是整数0,1,2

JvaScript中规定对象所有的键名都是字符串，而数组的键名也是字符串，只是javascript内部对数组对象的键名做了转换

上述代码数值和字符串都能都能获取到数组的值，原因是数值键名自动转换成了数组

而传统对象中获取值有两种方式，分别是 Object.key 或者 Object[key]

而数组就不能通过 arr[0] 来获取值

length属性

只要是数组，就一定会有length。length的值为最大下标值加1

上面代码说明数组的数字键不需要连续，length的值默认为最大下标+1

length的值是可写的，如果手动缩小length的长度，则之前保存的值会被删除

所以，有一种清空数组的方法，就是将 length的值修改为0

而当数组的length属性被修改为非法值时会报错

数组的本质是对象的一种，可以为数组添加属性，的那还是不会改变length的值
