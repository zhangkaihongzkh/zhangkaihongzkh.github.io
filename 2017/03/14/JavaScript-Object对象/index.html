JavaScript-Object对象

笔记摘自阮一峰老师的教程，当作复习细节以记之

Object对象

JavaScrpt原生提供一个Object对象，其他对象都继承自这个对象.Object本身也是一个构造函数，可以直接通过它来生成新对象

Object作为构造函数使用时，可以接受一个参数。如果该参数时一个对象，则返回这个对象。如果接受的参数是原始类型，则返回对应的包装类型

通过 new Object 方式形成的新对象，和字面量的形式 o = {} 是等价的

Object()

Object当作工具方法使用的时候，可以将任意值转化为对象

如果参数是原始类型的值,Object方法返回对应的包装对象的实例

如果Object方法是一个参数，总是返回原对象

Object对象的静态方法

静态方法就是部署在Object对象自身的方法

Object.keys(),Object.getOwnPropertyNames()

Object.keys()和Object.getOwnPropertyNames()都是用来遍历对象的属性

区别在于Object.keys()方法返回可枚举的属性，Object.getOwnPropertyNames()方法还返回不可枚举的属性名

当涉及不可枚举类型属性和的时候，两者就会有不一样的结果

Object对象的实例方法

Object实例对象，主要有以下6个

valueOf() :返回当前对象对应的值
toString() :返回当前对象对应的字符串的形式
toLocaleString() :返回当前对象对应的本地字符串形式。
hasOwnProperty() :判断某个属性是否为当前对象自身的属性，还是继承自原型对象的属性。
isPrototypeOf() :：判断当前对象是否为另一个对象的原型。
propertyIsEnumerable() :判断某个属性是否可枚举。

Object.prototype.valueOf()

valueOf方法的作用主要是返回一个对象的值，默认返回对象本身

valueOf方法的主要用途是，javascript在自动转化类型的时候会默认调用这个方法

Object.prototype.toString()

toString方法的作用是返回一个独享的字符串形式，默认情况下返回类型字符串

这个方法可以用于类型检测

数值：返回[object Number]。
字符串：返回[object String]。
布尔值：返回[object Boolean]。
undefined：返回[object Undefined]。
null：返回[object Null]。
数组：返回[object Array]。
arguments对象：返回[object Arguments]。
函数：返回[object Function]。
Error对象：返回[object Error]。
Date对象：返回[object Date]。
RegExp对象：返回[object RegExp]。
其他对象：返回[object Object]。
